Phoenix Arcade for the Altera DE2-35 Dev Board.

Notes:
Controls are PS2 keyboard, see readme file in de2 folder for instructions.

Build:
* Obtain correct roms file for phoenix, see make phoenix proms script in tools folder for rom filenames.
* Unzip rom files to tools/phoenix_unzip folder.
* Run the make phoenix proms script in the tools folder.
* Place the generated prom files inside the proms folder.
* Open the phoenix_de2 project file using Quartus and compile.
* Program DE2 Board.
